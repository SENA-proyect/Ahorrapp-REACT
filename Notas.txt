por si falla:
npm uninstall tailwindcss postcss autoprefixer
Remove-Item -Recurse -Force node_modules, package-lock.json

Instalacion:
npm install
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p <= si falla usar este comando: node .\node_modules\tailwindcss\lib\cli.js init -p
en caso de que no funcione, toca hacerlo manualmente

crear archivo:

postcss.config.js:
export default {
  plugins: {
    '@tailwindcss/postcss': {},
    'autoprefixer': {},
  },
}

instalar tambien esto: npm install @tailwindcss/postcss
